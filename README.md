# Gerenciador de Tarefas (Java + JUnit 5)

Projeto simples para praticar TDD e testes unitarios com JUnit 5.

## Estrutura

- `src/main/java/GerenciadorDeTarefas.java`: classe principal com metodos de negocio.
- `src/main/java/main.Main.java`: exemplo de execucao no terminal.
- `src/test/java/GerenciadorDeTarefasTest.java`: testes unitarios.

## Requisitos

- Java 17+
- Maven 3.9+

## Como executar os testes

```bash
mvn test
```

## Como executar o exemplo principal

```bash
mvn compile exec:java
```
